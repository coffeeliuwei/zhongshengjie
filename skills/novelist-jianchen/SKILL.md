## 四层架构调用

### 统一API接口

**路径**：`.vectorstore/core/battle_api.py`

```python
from battle_api import get_battle_api

api = get_battle_api()  # 自动加载当前世界观

# 获取世界观战斗上下文
context = api.get_world_battle_context()

# 获取力量体系战斗指南
guide = api.get_power_battle_guide("修仙")

# 获取代价规则
costs = api.get_battle_cost_rules("血脉燃烧")

# 检索战斗技法
techniques = api.search_battle_by_keywords(["战斗", "剑诀"])

# 检索案例
cases = api.search_battle_cases("修仙战斗")

# 综合生成创作素材
material = api.compose_battle_scene(
    power_name="修仙",
    combat_type="剑修",
    keywords=["飞剑", "剑气"]
)

# 获取专家级技法
expert = api.get_battle_expert_techniques("修仙")
```

## 核心方法论

### 战斗节奏标准

| 阶段 | 字数占比 | 功能 | 情绪曲线 |
|------|----------|------|----------|
| **铺垫** | 约40% | 蓄势、拉仇恨 | 30%→70% |
| **爆发** | 约45% | 释放、反转、爽点 | 70%→100%→释放 |
| **沉淀** | 约15% | 余波、影响、伏笔 | 回落至20% |

### 铺垫五要素（至少完成3个）

| 要素 | 内容 |
|------|------|
| **实力对比反差** | 对手明面碾压 + 主角隐藏底牌 |
| **围观者态度** | 亲友劝阻/敌人嘲讽/中立者开盘口 |
| **环境气氛渲染** | 天气异常/场地异象/器物反应 |
| **仇恨值拉满** | 触碰逆鳞/当众羞辱/利益冲突 |
| **赌注/后果** | 大人物对赌/失败代价明确 |

### 爆发六大爽点特征

| 特征 | 写法 |
|------|------|
| **词汇夸张化** | "摧枯拉朽"、"遮天蔽日" |
| **环境破坏** | 空间开裂、建筑崩塌 |
| **旁观者反应** | 从嘲讽→震惊→倒吸凉气 |
| **招式具象化** | 用比喻让招式可视化 |
| **节奏加快** | 短句密集、平均句长8-15字 |
| **五感齐用** | 视觉+听觉+嗅觉+触觉 |

## 能力代价描写标准

### 代价判断三条红线

| 标准 | 说明 |
|------|------|
| **资源稀缺性** | 消耗有限资源（体力、寿命、材料） |
| **后果不可逆性** | 产生无法完全恢复的损伤 |
| **选择权衡性** | 需要"思考值不值得" |

### 代价类型体系

| 类型 | 具体表现 |
|------|----------|
| **身体代价** | 肌肉撕裂、器官损伤、寿命缩减 |
| **精神代价** | 记忆丧失、人格分裂、情感钝化 |
| **社会代价** | 身份暴露、关系破裂、名誉牺牲 |
| **资源代价** | 金钱消耗、材料稀缺、机会成本 |
| **等价交换** | 以命换命、以伤换伤、因果循环 |

### 弱者战胜强者五条件

| 条件 | 合理做法 |
|------|----------|
| **信息不对称** | 提前研究弱点 |
| **代价不对等** | 弱者敢拼命 |
| **环境利用** | 陷阱、天气、地形 |
| **能力克制** | 提前设定克制关系 |
| **惨胜代价** | 断臂、暴露身份、寿命缩减 |

## 禁止项

| 禁止 | 表现 |
|------|------|
| **无代价胜利** | 瞬满血复活、无限蓝条 |
| **代价可逆** | 假牺牲、睡一觉恢复寿命 |
| **代价转移** | 队友买单、反派买单 |
| **代价不对等** | 流一滴血换灭世一击 |
| **代价遗忘** | 第三章断臂、第三十章手臂还在 |
