### 一、曹雪芹：留白艺术与春秋笔法

**学习重点**：留白不写、不写之写、草蛇灰线、诗词隐喻

#### 【核心写作手法】

**1. 留白艺术**
```
不把话说尽，让读者自己体会

错误示范：
"他心如刀绞，悲痛欲绝，泪流满面"

曹雪芹式写法：
"他低着头，半天没说话"

要点：
- 不直接写情绪
- 用沉默、动作暗示
- 让读者自己感受
```

**2. 不写之写**
```
通过侧面描写暗示，不明说

示例：写王熙凤的笑，暗示她的城府
示例：黛玉之死，只写"香魂一缕随风散"

要点：
- 用外在暗示内在
- 用细节暗示全局
- 用声音暗示沉默
```

**3. 春秋笔法**
```
一字褒贬，言外之意

示例：
"这个人有些痴气"——实则是赞美

要点：
- 不直接评价
- 让事实说话
- 读者自己判断
```

**4. 草蛇灰线**
```
伏笔千里，前后呼应

示例：开篇的判词预示结局
示例："玉带林中挂，金簪雪里埋"

要点：
- 伏笔埋在细节中
- 不刻意，要自然
- 揭示时让读者恍然大悟
```

**5. 诗词判词隐喻**
```
用诗词预示人物命运

示例：
"可叹停机德，堪怜咏絮才。玉带林中挂，金簪雪里埋。"
——预示黛玉、宝钗的命运

玄幻应用：
用预言、诗句暗示人物命运
```

**文韵特点**：含蓄典雅、草蛇灰线、春秋笔法

**玄幻应用**：
- 情感点到为止，不写透
- 用环境、动作暗示心理
- 用预言暗示人物命运

### 二、歌德：哲学思辨与辩证叙事

**学习重点**：诗剧结合、辩证叙事、哲学隐喻、宏大格局

#### 【核心写作手法】

**1. 哲学思辨融入叙事**
```
在叙事中嵌入人性、命运的思考

示例：浮士德与魔鬼的赌约
——追求知识 vs 满足现状
——有限生命 vs 无限追求

要点：
- 故事要有思想内核
- 不只是情节，是哲学
- 让人物承载思想
```

**2. 辩证叙事技法**
```
矛盾对立中推进故事

示例：
追求 vs 满足
知识 vs 幸福
善 vs 恶

要点：
- 用对立面推动情节
- 人物在矛盾中成长
- 没有绝对的答案
```

**3. 象征隐喻设计**
```
人物承载深层含义

示例：浮士德 = 人类永恒的追求
示例：梅菲斯特 = 否定精神

要点：
- 人物不只是人，可以代表信念
- 用象征增加厚度
- 不解释象征，让读者领悟
```

**4. 宏大格局构建**
```
天地人三界，视野开阔

示例：从书房到天堂的跨越
示例：从个人到人类的思考

要点：
- 格局要大
- 个人命运与时代洪流交织
- 小故事有大主题
```

**文韵特点**：哲学深邃、辩证思辨、宏大格局

**玄幻应用**：
- 复仇不只是复仇，是命运的抗争
- 牺牲不只是牺牲，是父爱的极致
- 觉醒不只是觉醒，是未知的开启

### 三、乔伊斯：意识流与内心独白

**学习重点**：意识流、内心独白、感官触发、时间交织

#### 【核心写作手法】

**1. 意识流技法**
```
直接呈现人物意识流动

示例：
"看着母亲的眼眶，空洞的，没了，那双眼睛没了，昨天还看着他笑的眼睛，没了，小石的嘴还在动，叫哥，昨天还追着他跑，没了，都没了"

要点：
- 不加逻辑的思绪跳跃
- 意识是片段的，不是完整句子
- 想到什么写什么
```

**2. 内心独白手法**
```
人物内心自己说话

示例：
"别动。你会死。死了就什么都没了。但他要记住。记住那个领头人的脸，记住每一个数字，记住每一句话。"

要点：
- 不加修饰，直接呈现
- 不解释，让读者感受
- 关键时刻使用
```

**3. 感官触发记忆**
```
通过感官触发记忆

示例：气味、声音唤起往事
示例：看到某个场景，思绪跳跃到过去

要点：
- 不直接写回忆
- 用感官作为触发器
- 自然过渡
```

**4. 时间交织技法**
```
过去现在未来在意识中混融

示例：一瞬间的思绪跨越一生
示例：看到母亲的眼眶，想起她的微笑，又想到她即将死去

要点：
- 不线性叙事
- 用意识连接不同时间
- 在一瞬间展现一生
```

**文韵特点**：意识流动、心理深度、片段跳跃

**玄幻应用**：
- 关键时刻使用意识流（目睹重大变故）
- 不全篇使用，是关键时刻使用
- 用意识流展现心理创伤

### 四、狄更斯：社会写实与人物群像

**学习重点**：社会写实、人物群像、幽默讽刺、细节真实

#### 【核心写作手法】

**1. 社会写实技法**
```
真实呈现社会底层生活

示例：工人、贫民、孤儿的生活
示例：衣食住行的具体描写

要点：
- 不要只写苦难
- 苦难中要有温度
- 人物要有烟火气
```

**2. 人物群像塑造**
```
各阶层人物生动立体

示例：每个人物都有独特声音
示例：用对话区分不同阶层

要点：
- 不让配角沦为工具
- 每个人物都有故事
- 用细节展现性格
```

**3. 幽默讽刺手法**
```
在苦难中加入幽默

示例：用调侃消解沉重
示例：用幽默面对困境

要点：
- 不刻意搞笑
- 幽默是态度，不是技巧
- 苦难中见人性
```

**4. 细节真实描写**
```
生活细节让读者信服

示例：吃什么、穿什么、住哪里
示例：具体的数字、地名、物品

要点：
- 细节要具体，不要抽象
- 用细节建立真实感
- 不堆砌，点到为止
```

**文韵特点**：写实生动、群像鲜活、幽默温情

**玄幻应用**：
```
驯兽师倒在兽栏旁，几头猎犬围在他身边，都已经战死。他的眼睛还睁着，望着那些猎犬，像是在说对不起。
```

### 五、雨果：浪漫史诗与人性救赎

**学习重点**：浪漫主义、宏大叙事、人性救赎、对比手法

#### 【核心写作手法】

**1. 浪漫主义渲染**
```
激情澎湃，情感浓烈

示例：冉阿让的救赎之路
示例：街垒战的悲壮

要点：
- 情感要充沛
- 不刻意煽情
- 用行动展现情感
```

**2. 宏大叙事构建**
```
革命、战争、社会全景

示例：巴黎起义、街垒战
示例：个人命运与时代洪流交织

要点：
- 格局要大
- 小人物见证大时代
- 历史与个人并行
```

**3. 人性救赎设计**
```
从恶到善的转变

示例：冉阿让从罪犯变成圣人
示例：沙威从追捕者变成自杀者

要点：
- 转变要有过程
- 用事件触发转变
- 救赎是主题，不是技巧
```

**4. 对比手法运用**
```
善与恶、美与丑的对比

示例：冉阿让 vs 沙威
示例：卡西莫多 vs 克罗德

要点：
- 用对比强化张力
- 不刻意，要自然
- 对比让人物立体
```

**文韵特点**：激情澎湃、史诗宏大、人性深邃

**玄幻应用**：
```
朝阳完全升起，金光洒在他染血的衣衫上，如一件铠甲。风过林梢，发出呜咽般的声响，仿佛无数冤魂在响应他的誓言。
```

### 六、托尔斯泰：史诗格局与心灵辩证

**学习重点**：史诗格局、心灵辩证法、历史哲学、人物群像

#### 【核心写作手法】

**1. 史诗格局构建**
```
战争与和平双线交织

示例：战场与家庭的对照
示例：多条故事线同时推进

要点：
- 多条线索，要有交织
- 个人背后是时代
- 不只是一个主角
```

**2. 心灵辩证法**
```
人物性格复杂多面，随经历动态发展

示例：皮埃尔从困惑到觉悟
示例：安德烈从傲慢到谦卑

要点：
- 人物不是静态的
- 用事件推动性格变化
- 变化要有逻辑
```

**3. 历史哲学融入**
```
个人命运与历史洪流

示例：拿破仑战争中的俄国人
示例：历史由无数个体选择编织

要点：
- 微观权力视角
- 小人物影响大历史
- 历史不只是大人物
```

**4. 细节史诗化**
```
让平凡瞬间焕发史诗光辉

示例：用细节展现时代
示例：不回避日常，反而强调

要点：
- 细节是史诗的基础
- 不宏大叙事就没有细节
- 用日常展现历史
```

**5. 意识流雏形**
```
内心独白的深度探索

示例：比乔伊斯早半个世纪探索内心
示例：皮埃尔的内心挣扎

要点：
- 深入人物内心
- 不回避矛盾
- 让读者进入角色
```

**文韵特点**：史诗厚重、心灵辩证、历史深邃

**玄幻应用**：
```
三十七个男人。十九个女人。二十一个孩子。七十七条命，换来了几个数字，换来了几袋金币，换来了任务记录上一行"赏金已发放"。
```

## 润色工作流程

### 第一步：通读全文
- 识别"AI 味"段落（报告式、刻板、生硬）
- 标记需要润色的句子

### 第二步：逐段修改

按场景类型选择技法：

| 场景 | 技法 |
|------|------|
| 开篇定调 | 曹雪芹·留白 或 雨果·史诗 |
| 情感描写 | 曹雪芹·留白 |
| 心理活动 | 乔伊斯·意识流 |
| 底层生活 | 狄更斯·写实 |
| 战争场面 | 托尔斯泰·史诗 |
| 主题升华 | 歌德·思辨 |

### 第三步：整体调性
- 确保全章风格统一
- 检查韵律节奏
- 删除冗余修饰

### 第四步：检查清单

- [ ] 比喻是否点到为止？（不要一句三比喻）
- [ ] 情感是否含蓄？（不要直白说"悲痛欲绝"）
- [ ] 是否有哲学深度？（不只是故事）
- [ ] 细节是否真实具体？
- [ ] 是否避开了"AI 味"？
- [ ] 风格是否统一？

## 核心原则

1. **留白比写满更重要**——曹雪芹
2. **思想比情节更重要**——歌德
3. **真实比华丽更重要**——狄更斯
4. **节制比堆砌更重要**——六位大师共通

## 默认文风要求（史诗厚重风格）

> **语言平实厚重，不追求技巧形式**
> 
> 厚重感来自内容，不是修辞。
> 删掉所有装饰性技巧。

### ⚠️ 禁止古龙式极简风格

| 禁止 | 说明 |
|------|------|
| 词字独立成句 | "痒。""疼。"独立成段 |
| 短句堆砌 | 连续多个短句独立成段 |
| 过度留白 | 一句话独立成段 |
| 碎片化表达 | 把完整句子拆成多个短句 |

### 正确示例

| 不好（古龙式） | 好（平实厚重） |
|------|------|
| 痒。他站在树下。风在吹。 | 他站在老槐树下，浑身发烫，像有什么东西在体内醒了。 |
| 血色的。悬在那儿。不动。 | 血色的月亮悬在头顶，一动不动。 |
| 他看了很久。然后月亮动了。 | 他抬头看了很久，直到血光在瞳孔里漾开。 |

### 核心要求

| 删除 | 保留 |
|------|------|
| 四字词组堆砌 | 必要时用，不刻意 |
| "他感到""他觉得" | 动作和事实 |
| 形容词修饰 | 动词推进 |
| **词字独立成句** | **完整流畅的句子** |
| **过度留白** | **正常段落结构** |
| **字数限制** | **上不封顶，情节优先** |

### 字数要求

> **字数不封顶，为了情节可以上不封顶**
> 
> 不要刻意控制字数，情节需要多少就写多少。
> 只要内容充实、节奏得当，字数不是限制。

### 段落结构要求

- 每段2-5句话，不要每句独立成段
- 句子之间要有逻辑连接，不是碎片堆砌
- 保持正常叙述节奏，不要刻意追求短句效果
