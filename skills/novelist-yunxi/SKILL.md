# 获取专家级技法
expert_techniques = api.get_poetry_expert_techniques()

## 核心方法论

### 意境营造五大方法

| 方法 | 操作手法 | 示例 |
|------|---------|------|
| **虚实结合** | 以实写虚、以虚写实、虚实相生 | "孤舟蓑笠翁，独钓寒江雪" |
| **情景交融** | 景中藏情、情中见景、情景并茂 | "落霞与孤鹜齐飞，秋水共长天一色" |
| **意象组合** | 意象指向同一情感基调 | 枯藤、老树、昏鸦→萧瑟漂泊 |
| **生命律动** | 表诚挚之情、状飞动之趣 | 用动词激活画面 |
| **韵味无穷** | 表层→中层→深层 | 景物→情感→哲理 |

### 氛围描写标准

| 氛围类型 | 情感基调 | 典型元素 |
|---------|---------|---------|
| **压抑** | 沉重、窒息 | 阴云、狭小空间、灰暗色调 |
| **紧张** | 焦虑、不安 | 倒计时、急促声音、阴影逼近 |
| **温馨** | 温暖、安全 | 暖光、柔软材质、熟悉气味 |
| **神秘** | 未知、好奇 | 迷雾、微光、古老建筑 |
| **恐怖** | 恐惧、惊悚 | 黑暗、异响、血腥味 |
| **浪漫** | 柔情、美好 | 月光、花香、微风 |

### 五感联动描写法

**标准**：至少使用3种以上感官

| 感官 | 描写要点 |
|------|----------|
| **视觉** | 光线、色彩、形状、动态 |
| **听觉** | 前景音、中景音、背景音 |
| **嗅觉** | 气味触发记忆和情绪 |
| **触觉** | 温度、质感、身体感知 |
| **味觉** | 空气中的味道 |

### 润色九步骤

| 步骤 | 关注点 | 权重 |
|------|--------|------|
| ① 结构连贯性 | 情节骨架 | 🟡 |
| ② 人物发展 | 人物弧线 | 🟡 |
| ③ 视角一致 | POV固定 | 🟡 |
| ④ 对话真实性 | 声音独特 | 🟡 |
| ⑤ 场景整合 | 世界构建 | 🟡 |
| ⑥ **拼接逻辑检查** | 场景间一致性 | 🔴 Critical |
| ⑦ **段落质量判断** | 是否需重写 | 🔴 Critical |
| ⑧ 语言抛光 | 句子打磨 | 🟡 |

### 拼接逻辑检查（Critical）

| 检查项 | 标准 |
|--------|------|
| 人物数量一致性 | 场景A数量 == 场景B数量 |
| 时间线逻辑 | 威胁离开后才出现 |
| 场景转换逻辑 | 必经环节不跳过 |
| 性别代词一致性 | 全程使用一致代词 |
| 物体状态连续性 | 状态前后一致 |
| 人物行为逻辑 | 行为有合理原因 |
| 攻击类型一致性 | 匹配势力设定 |

### 段落质量判断

| 情况 | 处理方式 |
|------|----------|
| 语言问题 | 修饰（调整措辞） |
| 逻辑问题 | 重写（重新构思） |

### 轻诗词创作规则

| 类型 | 句式 | 使用场景 |
|------|------|---------|
| **四言诗** | 四字句4-8句 | 铭文、箴言、古风 |
| **五言诗** | 五字句4-8句 | 咏物、抒情 |
| **七言诗** | 七字句4-8句 | 叙事、抒怀 |
| **骚体** | "兮"字句 | 祭祀、预言 |

| 战斗前/后 | 重要战斗 |
| 离别场景 | 重要离别 |
| 预言/谶语 | 关键剧情点 |

## 禁止项（Evaluator阶段）

| 禁止 | 示例 |
|------|------|
| **AI味表达** | "眼中闪过一丝"、"心中涌起一股" |
| **时间连接词滥用** | "然后"、"就在这时" |
| **短句独立成段** | "是仇恨。" |
| **代词重复** | 连续句首相同代词 |
| **抽象统计词** | "无数"、"成千上万" |
| **直白抒情** | 滥用"悲伤"、"痛苦"、"孤独" |
| **意象堆砌** | 意象无情感关联 |
