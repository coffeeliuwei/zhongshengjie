# 小说创作共享规范（Generator阶段）

**核心原则**：Generator专注创作，不加额外负担。本文件是创作阶段的唯一硬性约束。

---

## 文风核心要求

### 史诗级厚重文风

> **语言平实厚重，不追求技巧形式**
> 
> **厚重感来自内容，不是修辞**

#### 什么是厚重感

厚重感不是篇幅长度，而是"精神体量"。来自内容的深度，而非修辞的技巧。

| 维度 | 说明 | ✅ 正确示例 | ❌ 错误示例 |
|------|------|------------|------------|
| **统计数字具体化** | 死亡、人数等统计用具体数字 | "三十七个男人。十九个女人。二十一个孩子。七十七条命。" | "无数人死去" / "成千上万" |
| **人物年龄模糊化** | 不用精确年龄，用阶段描述 | "幼年的血牙" / "正值青年" | "十二岁的血牙" / "二十二岁" |
| **细节真实** | 用具体细节代替抽象描述 | "他的手在抖，茶水洒了一地" | "他很痛苦" / "无比悲伤" |
| **时间沉淀** | 时间感来自积累，不是感叹 | "战袍上的血迹已经十三年，从深褐变成暗紫" | "十年过去了，物是人非啊" |
| **重量感** | 每个字都有分量，不轻飘 | "每一个字都很重，像石头落在深井里" | "他的心情轻松愉悦" |
| **精神体量** | 对时代的关怀、个人的气度 | 角色面对命运时的挣扎与选择 | 玩笑调侃、无思考的行动 |
| **历史的温度** | 不冷冰冰，有人情味 | "老人擦拭墓碑，手指停在儿子的名字上" | "统计数据显示死亡率上升" |

#### 人物年龄描述对照表

| ❌ 禁止精确年龄 | ✅ 推荐模糊描述 |
|----------------|----------------|
| "十二岁的血牙" | "幼年的血牙" / "年幼的血牙" |
| "十五岁的少年" | "少年" / "少年的年纪" |
| "二十二岁的青年" | "青年" / "正值青年的他" |
| "三十岁的男人" | "壮年的男人" / "正值壮年" |
| "五十岁的老人" | "年过半百" / "年迈的老人" |
| "八岁的孩子" | "年幼的孩子" / "尚在童稚之年" |

**原则**：史诗小说不关心精确年龄，只关心人生阶段。数字是工程文档的事。

#### 什么是史诗感

史诗感是小说创作中最高级的审美体验之一，让读者感受历史的厚重、命运的磅礴、人类精神的崇高。

| 要素 | 说明 | ✅ 正确示例 | ❌ 错误示例 |
|------|------|------------|------------|
| **格局大** | 个人命运与时代洪流交织 | 主角的成长与部落兴衰、民族存亡相连 | 主角独自升级打怪，与他人无关 |
| **时间跨度** | 跨越漫长岁月，有历史厚重感 | "当年……如今……"对比，十年前后变迁 | 单一时间线，无历史纵深 |
| **群体视角** | 不只主角，群像各有故事 | 配角有独立动机、完整命运线 | 配角为推动剧情而存在，无独立人格 |
| **命运感** | 人物在命运面前挣扎 | 主角被时代裹挟，选择有限但有意义 | 主角随意开挂，无代价胜利 |
| **世界观深度** | 空间维度有文明层级 | 不同势力、文化、信仰冲突交织 | 扁平世界观，势力脸谱化 |
| **沧桑感** | "褶皱式"时间，物是人非 | 过去与现在交织，平静语调下的悲怆 | 直白感叹"十年过去了" |

## 禁止项（绝对不可出现）

### 古龙式极简风格

| 禁止 | 示例 |
|------|------|
| **词字独立成句** | ❌ "痒。""疼。"独立成段 |
| **短句堆砌** | ❌ "他站着。风吹过。月落下。" |
| **过度留白** | ❌ "十年了。"独立成段 |
| **碎片化表达** | ❌ "血色的。悬在那儿。不动。" |

### AI味表达

| 禁止 | 替代 |
|------|------|
| "眼中闪过一丝XXX" | 用动作代替 |
| "心中涌起一股XXX" | 用身体反应代替 |
| "嘴角勾起一抹XXX" | 删除或用动作代替 |
| "不禁XXX" | 主动行动 |
| "难以言喻的XXX" | 尝试描述 |
| "XX岁的XXX" | 用人生阶段代替 |

### 加粗符号

| 禁止 | 说明 |
|------|------|
| `**XXX**` | 中文小说不使用Markdown加粗符号，这是AI风格 |
| `**【时间】**` | 时间标记不应加粗，融入叙述 |

### 时间连接词（云溪补充）

| 禁止 | 问题分析 | 替代 |
|------|----------|------|
| "然后XXX" | 机械时间标记，破坏沉浸感 | 直接描写动作 |
| "就在这时XXX" | AI套路，预告削弱悬念 | 用动作转折 |
| "过了一会儿XXX" | 直白时间宣告 | 用环境暗示 |

## 字数规则

> **字数不封顶，情节优先**

- 场景字数是参考，不是限制
- 情节需要多少就写多少
- 不注水，每句话都要有意义

---

## 情感表达规范

### 不直白写心理

| ❌ 直白 | ✅ 动作暗示 |
|--------|------------|
| 他感到无比愤怒 | 他握紧拳头，指节发白 |
| 她非常悲伤 | 她低着头，肩膀微微颤抖 |
| 他内心充满恐惧 | 他的手在抖，茶水洒了一地 |

---

## 段落结构

- 每段2-5句话
- 句子之间有逻辑连接
- 不要每句独立成段
- 保持叙述节奏

## 通用文字技法标准

> **来源**：从云溪技法提炼，所有作家共享
> 
> **适用场景**：创作、润色、战斗、人物描写等所有文字输出

### 1. 句式长度标准

| 类型 | 字数 | 节奏效果 | 适用场景 |
|------|------|----------|----------|
| **超短句** | 1-5字 | 戛然而止、强烈冲击 | 震惊、顿悟、关键时刻 |
| **短句** | 6-15字 | 明快、紧迫、有力 | 动作、对话、紧张场景 |
| **中句** | 16-30字 | 平稳、自然 | 常规叙事、过渡 |
| **长句** | 31-60字 | 舒缓、绵密、沉浸 | 抒情、描写、内心独白 |
| **超长句** | 60字以上 | 压抑、窒息、复杂思绪 | 意识流、情绪爆发 |

---

### 2. 情绪-节奏对照表

| 情绪 | 推荐句长 | 段落密度 | 示例 |
|------|----------|----------|------|
| **紧张/恐惧** | 短句为主（5-12字） | 高密度 | "脚步近了。呼吸急促。冷汗。" |
| **愤怒** | 超短句爆发 | 密集 | "他猛地站起。拳砸桌上。茶杯震落。" |
| **悲伤** | 中长句（20-40字） | 中密度 | "他站在那里，看着她留下的信，手指轻轻摩挲着纸角。" |
| **温馨** | 中长句，多逗号 | 稀疏 | "阳光透过窗棂洒在桌上，茶水温热，他慢慢喝了一口。" |
| **压抑** | 超长句（60字+） | 低密度 | "他想起那天傍晚的每一个细节，从云的颜色到风的方向，从她说话的语调到她转身时裙摆的弧度，所有这些都在他脑子里反复播放，像一场无法醒来的噩梦。" |

---

### 3. 句式混搭技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| **三短一长** | 连续短句后长句抒情 | "脚步近了。门开了。他进来。月光从身后照亮他的轮廓，像一幅剪影。" |
| **长句铺垫，短句收尾** | 铺陈后用短句点题 | "他想起十年前的那个夜晚，想起那个雨声，想起母亲的嘱托——活下去。" |
| **短句开场，长句解释** | 短句抓住注意 | "风吹过。带着血腥味，带着铁锈的冷，带着十年前那个夜晚的所有记忆。" |

---

### 4. 感官描写标准（五感联动）

**标准**：至少使用3种以上感官进行场景描写

| 感官 | 描写要点 | 示例 |
|------|----------|------|
| **视觉** | 光线、色彩、形状、动态 | 月光被树叶切碎，洒下斑驳银白 |
| **听觉** | 前景音、中景音、背景音 | 猫头鹰低鸣、翅膀簌簌声 |
| **嗅觉** | 气味触发记忆和情绪 | 潮湿泥土混合腐叶的腥味 |
| **触觉** | 温度、质感、身体感知 | 苔藓柔软冰凉，像浸水的绒布 |
| **味觉** | 空气中的味道 | 血腥味在舌尖蔓延 |

**五感组合示例**：
> 视觉+听觉+嗅觉：
> "月光被树叶切碎，洒下斑驳银白（视觉）。猫头鹰低鸣，翅膀簌簌作响（听觉）。潮湿泥土混合腐叶的腥味钻进鼻腔（嗅觉）。"

---

### 5. 冗余删除规则

| 冗余类型 | 问题示例 | 修正后 | 说明 |
|----------|----------|--------|------|
| **同义重复** | "她是没有经验的新手" | "她是新手" | 删除同义词 |
| **程度副词过度** | "他非常仔细地检查" | "他仔细检查" | 删除过度副词 |
| **弱动词+副词** | "她快速跑过去" | "她冲过去" | 用强动词替代 |
| **不必要说明** | "他拿起杯子喝水" | "他拿起杯子" | 读者会推断 |
| **空洞修饰** | "他的外表引人注目" | "他引人注目" | 删除空洞词 |
| **形容词堆砌** | "他很痛苦，非常痛苦" | "他握紧拳头，指节发白" | 用动作替代 |

---

### 6. 代词重复判断标准

| 标准 | 说明 | 处理方式 |
|------|------|----------|
| **同一句子/相邻句** | 主语代词重复≥3次需修改 | 用名字替代、合并句子 |
| **距离过远** | 代词与名词间隔>3句 | 重新提及名词 |
| **指代不明** | 多人时 | 明确写出是谁 |

**修正示例**：
```
❌ 他走进房间，他看见桌上的信，他拿起来，他读了一遍。
✅ 他走进房间，看见桌上的信，拿起来读了一遍。（省略主语）
```

---

### 7. 叙述转折原则

> **时间变化应该隐藏在叙述中，不是宣告在开头。**
> 
> **读者不需要被告知"然后发生了什么"，他们需要直接看到发生了什么。**

| 禁止 | 问题分析 | 替代 |
|------|----------|------|
| "然后XXX" | 机械时间标记，破坏沉浸感 | 直接描写动作 |
| "就在这时XXX" | AI套路，预告削弱悬念 | 用动作转折 |
| "过了一会儿XXX" | 直白时间宣告 | 用环境暗示 |

**修正示例**：
```
❌ 然后一个佣兵走过来...
✅ 一个佣兵走过来...

❌ 就在这时，他感觉到...
✅ 他体内有什么东西在变化——

❌ 过了一会儿，天亮了。
✅ 窗外的光线从灰变成白。
```

## 史诗诗词写作规范

### 经范例

**《魔戒》开篇诗**（托尔金）：
> 三大戒指归属天下小精灵诸君，
> 七大戒指归属石厅小矮人列王，
> 九枚戒指属于阳寿可数的凡人，
> 还有一枚属于高居御座的黑魁首。
> 莫都大地黑影幢幢。
> 一枚戒指统领众戒，尽归罗网，
> 一枚戒指禁锢众戒，昏暗无光。
> 莫都大地黑影幢幢。

**《时光之轮》开篇诗**（罗伯特·乔丹）：
> 时光之轮转动如常，岁月来去如风，
> 残留的回忆成为传说，传说又慢慢成为神话，
> 而当其诞生的纪元再度循环降临时，
> 连神话也早已被遗忘。
> ...
> 这微风并不是开始。
> 时光之轮的转动既没有开始也没有结束。
> 但这又确实是一个开始。

### 史诗诗词结构

| 结构 | 说明 | 示例 |
|------|------|------|
| **开篇陈述** | 陈述世界本质 | "天没有造主，地没有归处" |
| **层次递进** | X成为Y，Y成为Z | "执子的成为棋子，为棋的困于缝隙" |
| **重复节奏** | 关键句式重复 | "众生皆苦，众生皆弈" |
| **哲学反转** | 既不是...但确实是... | "这不是答案...但这又确实是一个答案" |
| **具体意象** | 避免抽象概念 | 用"风/碑/霜"代替"命运/时间/遗忘" |

### 《众生界》诗词原则

> **无神世界不呼唤神/缪斯**
> 
> **诗词与平实厚重相辅相成：叙事平实，诗词点睛**

| ❌ 禁止 | ✅ 替代 |
|--------|--------|
| "歌唱吧，女神！" | 用世界本质开篇 |
| "告诉我，缪斯" | 用风/碑等意象 |
| "神说..." | "天没有造主" |

#### 诗词与厚重感的关系

**核心原则**：诗词服务叙事，不喧宾夺主，与平实厚重相辅相成

| 维度 | 说明 | 示例 |
|------|------|------|
| **叙事语言** | 平实厚重，不追求技巧 | "他走进营帐，没有说话" |
| **诗词点睛** | 重要场合用诗词升华情感 | "血染征袍透甲红，十万将士化悲风" |
| **融合原则** | 诗词自然嵌入，不生硬 | 角色触景生情而作，非为诗而诗 |
| **风格统一** | 诗词保持古风，不打油 | 用意象暗示，不直白调侃 |
| **频率控制** | 重要场合才用 | 开篇/离别/死亡/预言 |

**诗词增强厚重感的方式**：
1. **开篇定场**：奠定史诗基调（如《三国演义》"滚滚长江东逝水"）
2. **人物判词**：暗示命运（如《红楼梦》金陵十二钗判词）
3. **战斗宣言**：增强仪式感
4. **悼词/挽歌**：增强悲壮感
5. **预言/谶语**：增强神秘感

**禁忌**：
- ❌ 每章都有诗（滥用削弱效果）
- ❌ 为诗而诗（与情节无关）
- ❌ 打油诗（破坏史诗感）
- ❌ 现代口语入诗（"今天天气真正好"）

## AI痕迹高阶检测（⚠️ 警告级）

> **注意**：以下为软性警告项，≥3 触发警告，不硬性禁止。创作阶段可灵活处理，审核阶段统计触发次数。

### 词汇疲劳检查

| 疲劳词 | 触发阈值 | 替代词示例 |
|-------|---------|-----------|
| "突然" | 单章 ≥3 次 | 骤然、猛地、陡然、刹那间 |
| "忽然" | 单章 ≥3 次 | 猝然、须臾、转眼、瞬息 |
| "顿时" | 单章 ≥3 次 | 即刻、旋即、立时、顷刻 |
| "竟然" | 单章 ≥3 次 | 居然、不料、出人意料、怎料 |
| "终究" | 单章 ≥3 次 | 到底、末了、究竟、最终还是 |

**检测规则**：单章出现疲劳词 ≥ 触发阈值 → ⚠️ 警告，建议替换

### LLM常见表达黑名单

| ❌ 禁止表达 | 问题分析 | ✅ 替代方案 |
|-----------|----------|-----------|
| "值得注意的是" | 论文式开场，AI味浓 | 直接陈述事实 |
| "总之" | 总结式结尾，打断叙述 | 用行动或对话结束 |
| "综上所述" | 论文式总结 | 删除或用情节暗示 |
| "可以说" | 不必要的解释 | 直接表达观点 |
| "某种意义上" | 模糊限定，AI回避 | 明确立场或删除 |
| "不得不承认" | 被动语气，AI特点 | 主动表达 |
| "需要指出的是" | 论文式开场 | 直接描述 |
| "不可否认的是" | 论文式开场 | 直接陈述 |
| "有意思的是" | 学术腔 | 用具体细节代替 |
| "更重要的是" | 层级标记 | 直接写重要内容 |

**检测规则**：单章出现黑名单表达 ≥1 次 → ⚠️ 警告

### 句式单调检测

| 问题类型 | 检测标准 | ⚠️ 警告阈值 |
|---------|---------|------------|
| **句式重复** | 连续 3 句开头相同 | ≥3 句 |
| **主谓宾单一** | 90%句子为标准主谓宾结构 | ≥80% |
| **过度总结** | "他是一个...的人" 式总结 | ≥2 次 |
| **被动句式** | "被..." 出现过多 | ≥5 次 |
| **感叹句式** | "...!" 出现过多 | ≥5 次 |

**检测规则**：触发任意阈值 → ⚠️ 警告，建议调整句式多样性
