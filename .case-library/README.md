## 三、提取流程

### 3.1 阶段一：格式预处理

```
输入：E:\小说资源\ （3960个文件）
├── epub: 2361个（需转换）
├── mobi: 959个（需转换）
└── txt: 639个（可直接使用）

处理：
├── epub/mobi → 转换为txt（使用 ebooklib）
├── 统一编码为UTF-8
├── 清理格式标记
└── 按章节分割（可选）

输出：
└── converted/*.txt（转换后的文件）
```

#### 3.1.1 格式转换命令

```bash
# 转换指定目录
python convert_format.py --dir "E:\小说资源\玄幻奇幻"

# 转换所有小说资源（默认）
python convert_format.py

# 不递归处理子目录
python convert_format.py --dir "E:\小说资源" --no-recursive

# 只转换 epub
python convert_format.py --formats epub
```

#### 3.1.2 转换状态

| 格式 | 数量 | 依赖 | 状态 |
|------|------|------|------|
| epub | 2361 | ebooklib | ✅ 可转换 |
| mobi | 959 | calibre | ⚠️ 需安装calibre |
| txt | 639 | 无需转换 | ✅ 可直接使用 |

> **注意**: mobi 转换需要安装 Calibre 软件。建议优先处理 epub 和已有的 txt 文件。

### 3.2 阶段二：场景识别

```
方法：Agent批量分析

对每本小说：
├── 识别章节结构
├── 标记场景类型（开篇/战斗/对话/情感等）
├── 识别关键人物出场
├── 标记关键转折点
└── 输出场景索引JSON

场景索引JSON示例：
{
  "novel": "盘龙",
  "chapters": [
    {
      "chapter": 1,
      "scenes": [
        {"type": "开篇", "start": 0, "end": 856},
        {"type": "人物出场", "start": 1200, "end": 1500, "character": "林雷"},
        {"type": "战斗", "start": 3500, "end": 4200}
      ]
    }
  ]
}
```

### 3.3 阶段三：片段提取

```
根据场景索引，提取目标片段：

提取规则：
├── 开篇场景：章节开头500-1000字
├── 人物出场：人物首次出现的完整段落
├── 战斗场景：完整的战斗描写（从开始到结束）
├── 对话场景：有意义的对话片段（200-500字）
├── 情感场景：情感表达的关键段落
├── 悬念场景：悬念设置的核心片段
├── 转折场景：转折前后的完整段落
└── 结尾场景：章节最后300-500字

质量过滤：
├── 片段完整性（不是断裂的）
├── 语言质量（无AI味、无语病）
├── 技法体现（有明显技法应用）
└── 代表性（能代表该类型的特点）
```

### 3.4 阶段四：质量评估

```
每个案例评估维度：

| 维度 | 说明 | 权重 |
|------|------|------|
| 完整性 | 片段是否完整 | 20% |
| 技法体现 | 技法应用是否明显 | 30% |
| 代表性 | 能否代表该类型 | 25% |
| 语言质量 | 无禁止项、无AI味 | 25% |

评分标准：
├── 8-10分：优秀案例，强烈推荐
├── 6-7分：良好案例，推荐
├── 4-5分：一般案例，可选
└── 0-3分：不推荐
```

## 十二、新题材自动识别与注册（新增）

### 12.1 新题材检测机制

```
当新增小说时，系统自动检测题材类型：

检测流程：
├── 优先级1：数据源定义的题材（sources.json）
├── 优先级2：目录名/文件名特征匹配
├── 优先级3：内容关键词分析
└── 兜底：新题材检测

新题材判定条件：
├── 所有已知题材得分都很低（< 阈值）
├── 内容中存在独特的特征词组合
└── 至少提取到5个以上高频特征词

自动提取特征词：
├── 统计内容中的高频词语（2-4字）
├── 排除已知题材特征词
├── 排除通用词（的/是/在等）
└── 提取前20个作为候选特征
```

### 12.2 新题材注册流程

```
系统发现疑似新题材 → 记录到 new_genres.json

用户操作步骤：
1. python incremental_sync.py --list-candidates
   → 查看待确认的新题材列表

2. python incremental_sync.py --register-genre "悬疑推理" --features "推理,破案,侦探,线索"
   → 注册新题材

注册后自动更新：
├── config.json → genres 列表新增题材
├── genre_classifier.py → GENRE_KEYWORDS 新增关键词
├── novel_index.json → 已标记小说更新题材
└── new_genres.json → candidate标记为confirmed
```

## 十五、跨题材提取策略（核心设计）

### 15.1 设计理念

```
《众生界》是史诗级玄幻小说，应该跨题材吸收精华：

核心原则：
├── 不局限于玄幻题材
├── 从各题材中提取可借鉴的叙事技法
├── 按场景类型组织，而非题材
└── 标注来源题材，便于理解跨题材借鉴价值
```

### 15.2 跨题材借鉴矩阵

| 来源题材 | 贡献场景 | 借鉴价值 |
|----------|----------|----------|
| **玄幻/仙侠** | 修炼突破、世界观构建、战斗、资源 | 玄幻核心技法，力量体系构建 |
| **历史军事** | 势力登场、权谋博弈、战争场面 | 史诗氛围，势力构建深度 |
| **悬疑推理** | 伏笔设置、悬念营造、反转设计 | 长线叙事张力，剧情戏剧性 |
| **现代都市** | 对话技巧、情感细腻、心理刻画 | 人物互动质量，情感线深度 |
| **女频言情** | 情感层次、心理描写、人物刻画 | 感情线处理，情感细腻度 |
| **科幻灵异** | 世界观宏大、悬念设置、未知探索 | 世界观尺度，未知恐惧感 |
| **武侠仙侠** | 战斗描写、江湖氛围、侠义精神 | 招式细腻度，氛围渲染 |
| **青春校园** | 成长弧光、心理细腻、情感真挚 | 主角成长，心路历程 |
| **游戏竞技** | 升级节奏、成就感设置、团队配合 | 升级体验，资源获取节奏 |

### 15.3 场景类型最佳来源

| 场景类型 | 最佳来源题材 | 跨题材价值 |
|----------|--------------|------------|
| 开篇场景 | 玄幻、科幻、悬疑 | 不同开篇风格多样性 |
| 战斗场景 | 玄幻、武侠、历史 | 武侠招式提升玄幻战斗细腻度 |
| 对话场景 | 都市、言情、历史 | 都市对话技巧提升互动质量 |
| 情感场景 | 言情、校园、都市 | 言情情感细腻提升玄幻情感线 |
| 悬念场景 | 悬疑、科幻、玄幻 | 悬疑技法是玄幻剧情张力核心 |
| 转折场景 | 悬疑、玄幻、历史 | 悬疑反转设计提升戏剧性 |
| 心理场景 | 都市、言情、校园 | 都市心理描写丰富人物内心 |
| 势力登场 | 历史、玄幻、武侠 | 历史势力构建提升世界观深度 |
| 修炼突破 | 玄幻、武侠 | 玄幻核心，需大量案例 |
