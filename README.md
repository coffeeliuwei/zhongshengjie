# 众生界

> 千山不现路难寻  
> 万象空庭棋自沉  
> 执子终成局中子  
> 风碑无声证天涯

---

## 简介

天无主，地无归处。

千年的博弈从未停歇，岁月来去如风，执子之人终化作棋子，被棋局困在缝隙之间；缝隙里的他们，早把名字尘封。

众生皆苦，众生皆弈，众生在追问：我是谁？

无人应答的沉默里，风穿过石碑的纹路，穿过荒野的盐霜，穿过黎明前无人铭记的日落。

这不是答案。千年的追问，既无答案，也无尽头；却如同一枚未落定的棋子，静默地宣布自己的存在。

去问风，去问碑，去问那些死在黎明前的人。他们曾以为自己知道。

## 世界观

七大力量体系交织：修仙、魔法、神术、科技、兽力、AI力、异能。

十大势力割据天下，各有立场，无正无邪。

众生在时代洪流中追问身份，寻找归属。
